#DSA 
#Sorting
#Complexity

n = number of elements
k = length of range of values
d = number of digits of an element
b = base (10, 16, 2, 8, ...)

Selection Sort
O(n²)

Merge Sort
O(nlogn)

Heap Sort
O(nlogn)

Quick Sort
O(nlogn)

Counting Sort
O(n+k)

Bucket Sort
O(n+k)

Radix Sort
O(d(n+b))

Shell Sort
O(n(logn)²)
